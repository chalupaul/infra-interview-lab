# Infrastructure Automation Interview Lab

This repo contains two independent hands-on labs. Your interviewer will tell you which one to start with.

| Lab | Directory | Stack |
|---|---|---|
| Terraform + CI/CD + Policy | `terraform/` | Terraform, GitHub Actions, OPA/Conftest |
| Python resource processor | `python/` | Python 3, pytest |

See the `README.md` inside each directory for instructions.
